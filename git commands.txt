1.Setup your username and email:
git config --global user.name "Karan"
git config --global user.email "karanpillai721@gmail.com"

2.Initialise git
git init

3.Check status
git status

4.Add files to staging area
git add --a/ git add .

5.Remove .git file which will delete complete data
rm -rf .git

6.To ignore any files:
touch .gitignore
add the file which u want to ignore in that gitignore file.

7.git diff
Compare staging and working directory

8.Removing files through git(It will remove the file and add it to staged area)
git -rm filename.extension
